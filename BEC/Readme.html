Para usar este control personalizado en un archivo XAML en otro proyecto, complete los pasos siguientes:

  • Agregue una referencia a este proyecto.
  • Agregue la siguiente línea al elemento raíz del archivo XAML donde desea usar este control:
    xmlns:MyNamespace="clr-namespace:BEC;assembly=BEC"
  • Usar el control en el archivo XAML:
    <MyNamespace:CustomControl1/>
